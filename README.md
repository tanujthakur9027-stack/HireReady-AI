# HireReady 

> **AI-Powered Career Preparation Platform**

HireReady is an intelligent interview preparation platform designed to help candidates improve their resumes, optimize ATS compatibility, practice AI-powered mock interviews, receive coding feedback, and track interview performance through analytics and persistent memory.

---

##  Features

-  AI Resume Analysis
-  ATS Score Evaluation
-  Resume vs Job Description Matching
-  AI Mock Interviews
-  Coding Interview Practice & Review
-  Voice-Based Interview Support
-  Camera Integration (Interview Mode)
-  Interview Analytics & Weak Topic Tracking
-  Persistent Candidate Memory (Cognee)
-  PDF Report Generation
